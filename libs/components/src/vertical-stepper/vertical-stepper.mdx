## Vertical Step

The `<bit-vertical-step>` component extends the
[Angular CdkStep](https://material.angular.io/cdk/stepper/api#CdkStep) component

The following additional Inputs are accepted:

| Input         | Type    | Description                                                    |
| ------------- | ------- | -------------------------------------------------------------- |
| `subLabel`    | string  | An optional supplemental label to display below the main label |
| `applyBorder` | boolean | Show/hide the decorative border connecting the steps           |

In order for the stepper component to work as intended, it's children must be type of
`VerticalStep[]`.

```html
<bit-vertical-stepper #stepper>
  <bit-vertical-step label="This is the label" subLabel="This is the sub label">
    Your content here
  </bit-vertical-step>
  <bit-vertical-step label="Another label"> Your content here </bit-vertical-step>
  <bit-vertical-step label="The last label"> Your content here </bit-vertical-step>
</bit-vertical-stepper>
```
