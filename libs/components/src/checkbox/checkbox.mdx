## Using in a group

Use `bit-form-control-group` when you have multiple related checkboxes that should be validated
together as a single form field. The group binds to a `string[]` form control — each card's
`[value]` input is added to or removed from the array when checked.

Choose checkboxes in a group when:

- The user selects from a set of options that are saved via a form submit action
- At least one selection should be required (`Validators.required` on the group control enforces
  this)
- The options are independent of each other (any combination is valid)

Use switches instead when each option should take effect immediately without a save action.
